Comment ça fonctionne ?
  • La synchronisation récupère les compétitions depuis WECANPRONO-SERVICE via le gateway
  • Les compétitions sont créées/mises à jour dans la base locale
  • Un scheduler automatique synchronise toutes les 6 heures (configurable)
  • Vous pouvez déclencher une synchronisation manuelle avec le bouton ci-dessus
  • Les compétitions synchronisées sont ensuite disponibles pour créer des phases et des matches
