Conseils
  • Code : Utilisez un code court et reconnaissable. Exemples : PL (Premier League), FL1 (Ligue 1), UCL (Champions League)
  • Unicité : Le code doit être unique dans la base de données. Si vous créez une compétition qui existe déjà dans WECANPRONO, utilisez la synchronisation.
  • Synchronisation : Pour les compétitions officielles, préférez utiliser la fonction de synchronisation plutôt que la création manuelle.
  • Pays : Indiquez le pays pour les compétitions nationales, ou "Europe", "International" pour les compétitions multi-pays.
Exemples de Codes Courants
  • PL - Premier League (England)
  • FL1 - Ligue 1 (France)
  • LL - La Liga (Spain)
  • SA - Serie A (Italy)
  • BL - Bundesliga (Germany)
  • UCL - UEFA Champions League
  • UEL - UEFA Europa League
  • CAN - Coupe d'Afrique des Nations
  • WC - World Cup
  • EC - European Championship
